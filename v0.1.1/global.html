Type Definitions

Options

Request options understood by this package. All other options will be passed through to m3api.

It’s strongly recommended to use the same options for #login and #logout, typically by including them in the session’s defaultOptions at construction time. Using different options, especially logging in with an option set to true and then logging out with it set to false, will likely result in all future requests failing with an 'assertuserfailed' error.
